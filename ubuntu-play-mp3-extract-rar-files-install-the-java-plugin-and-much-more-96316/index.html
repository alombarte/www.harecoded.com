Ubuntu: Play mp3, extract RAR files, install the java plugin... and much more

Sun, Apr 2, 2006

Recentlly I posted on How to play mp3 with ubuntu and now I am posting how to do it again but also:

  • How to install the Java Plugin
  • How to install the Flash Plugin
  • How to play windows videos
  • RealPlayer
  • Extract RAR files, ACE files and 7-Zip files
  • Enable the Num Lock at system startup
  • Install microsoft fonts
  • and much much more...

The solution arrives by installing EasyUbuntu

Run this code into the terminal and enjoy :)

sudo apt-get install subversion
cd ; svn checkout svn://freecontrib.org/easyubuntu
gksudo easyubuntu/trunk/easyubuntu.py
